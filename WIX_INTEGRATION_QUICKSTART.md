# 🚀 Wix連携クイックスタートガイド（5分で完了）

## ✨ 最も簡単な方法：ボタンリンク

### 📌 手順（5ステップ、所要時間: 約5分）

---

### **Step 1: Wixにログイン** (1分)

1. https://www.wix.com/ にアクセス
2. ログイン
3. 「**サイトを編集**」をクリック

---

### **Step 2: ボタンを追加** (1分)

1. 左側の「**追加 (+)**」をクリック
2. 「**ボタン**」を選択
3. お好みのデザインを選択
4. ページに配置

---

### **Step 3: ボタンテキストを変更** (30秒)

ボタンのテキストを以下のいずれかに変更：

```
✅ 推奨テキスト:
- 製品カタログを見る
- 製品一覧
- Products
- View Catalog
```

---

### **Step 4: リンクを設定** (1分)

1. ボタンをクリック
2. 左側の「**リンク**」アイコン🔗をクリック
3. 「**外部リンク**」を選択
4. URL欄に入力:

```
https://app.fvnix.com/
```

5. ✅「**新しいタブで開く**」にチェック
6. 「**完了**」をクリック

---

### **Step 5: 公開** (30秒)

1. 右上の「**公開**」ボタンをクリック
2. ✅ 完了！

---

## 📱 メニューにも追加（オプション、+2分）

### 手順:

1. **トップメニュー**をクリック
2. 「**メニューを管理**」を選択
3. 「**+** メニュー項目を追加」をクリック
4. 項目名: **製品カタログ**
5. リンクタイプ: **外部リンク**
6. URL: `https://app.fvnix.com/`
7. ✅「**新しいタブで開く**」を有効化
8. 「**保存**」をクリック

## 🎨 ボタンデザインのヒント

### おすすめスタイル:

- **色**: ブランドカラー（目立つ色）
- **サイズ**: 中〜大サイズ
- **形**: 角丸長方形
- **テキスト**: 明確なアクション（「〜を見る」「〜する」）

### 良い例 ✅:
```
[製品カタログを見る →]
[View Products →]
[詳細はこちら]
```

### 避けるべき例 ❌:
```
[こちら]
[Click]
[More]
```

## 🔗 設定するURL

```
✅ 正しいURL:
https://app.fvnix.com/

❌ 間違ったURL:
http://app.fvnix.com/  (httpではなくhttps)
app.fvnix.com  (httpsが必須)
www.app.fvnix.com  (wwwは不要)
```

## 🆘 トラブルシューティング

### Q: ボタンをクリックしても何も起こらない

**A:** リンク設定を確認してください
- ボタンをクリック → リンクアイコン → URLが正しく入力されているか確認

### Q: 404エラーが表示される

**A:** URLを確認してください
- ✅ 正: `https://app.fvnix.com/`
- ❌ 誤: `http://app.fvnix.com/`

### Q: リンクが同じタブで開いてしまう

**A:** 「新しいタブで開く」設定を有効化してください
- リンク設定 → 「新しいタブで開く」にチェック
