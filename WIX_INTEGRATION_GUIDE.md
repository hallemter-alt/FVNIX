## 方法1: Wixボタン/メニューからリンク ⭐ 推奨

### ステップ1: Wix Editor にログイン

1. https://www.wix.com/ にアクセス
2. ログインして「サイトを編集」をクリック
3. Wix Editor を開く

### ステップ2: ボタンを追加/編集

#### A. 既存のボタンを編集する場合

1. **ボタンをクリック**して選択
2. 左側のパネルで「**リンク**」アイコンをクリック
3. 「**外部リンク**」を選択
4. URL欄に入力: `https://app.fvnix.com/`
5. 「**新しいタブで開く**」にチェック（推奨）
6. 「**完了**」をクリック

#### B. 新しいボタンを追加する場合

1. 左側パネルで「**追加 (+)**」をクリック
2. 「**ボタン**」を選択
3. ボタンのデザインを選択
4. ボタンのテキストを変更: 例）「**製品カタログを見る**」「**Products**」
5. 上記A の手順でリンクを設定

### ステップ3: メニュー（ナビゲーション）にリンクを追加

1. **トップメニュー**をクリック
2. 「**メニューを管理**」を選択
3. 「**メニュー項目を追加**」をクリック
4. 項目名: 「**製品カタログ**」または「**Products**」
5. リンクタイプ: 「**外部リンク**」
6. URL: `https://app.fvnix.com/`
7. 「**新しいタブで開く**」を有効化（推奨）
8. 「**保存**」をクリック

### ステップ4: テキストリンクを追加

1. テキストボックスを選択
2. リンクを設定したいテキストを選択（例: "詳しくはこちら"）
3. ツールバーの「**リンク**」アイコンをクリック
4. 「**外部リンク**」を選択
5. URL: `https://app.fvnix.com/`
6. 「**完了**」をクリック

### ステップ5: 公開

1. 右上の「**公開**」ボタンをクリック
2. 変更を公開

## 方法2: Wix Embed（iframe）で埋め込み

⚠️ **注意**: この方法はデザインの制御が難しく、パフォーマンスに影響する可能性があります。

### 手順:

1. Wix Editorで「**追加 (+)**」をクリック
2. 「**埋め込み**」→「**HTMLコード**」を選択
3. 「**コード**」タブをクリック
4. 以下のコードを貼り付け:

```html
<iframe 
  src="https://app.fvnix.com/" 
  width="100%" 
  height="800px" 
  frameborder="0"
  style="border: none;">
</iframe>
```

5. 「**適用**」をクリック
6. サイズと位置を調整
7. 「**公開**」

### デメリット:

- ❌ SEOに不利
- ❌ モバイル対応が難しい
- ❌ ページ読み込みが遅くなる可能性
- ❌ ナビゲーションが複雑になる

## 方法3: Wix Velo（開発者向け）

Wix Veloを使用して、動的なリンクやルーティングを実装できます。

### 基本例:

```javascript
// Wix Velo コード例
$w.onReady(function () {
  // ボタンクリック時にCloudflare Pagesへリダイレクト
  $w("#productsButton").onClick(() => {
    wixLocation.to("https://app.fvnix.com/");
  });
});
```
