## 🛠️ ローカル開発

### インストール

```bash
cd /home/user/webapp
npm install
```

### ビルド

```bash
npm run build
```

### 開発サーバー起動（PM2使用）

```bash
# ポート3000をクリーンアップ
fuser -k 3000/tcp 2>/dev/null || true

# PM2で起動
pm2 start ecosystem.config.cjs

# ログ確認
pm2 logs fvnix-webapp --nostream

# PM2停止
pm2 delete fvnix-webapp
```

### ローカルテスト

```bash
curl http://localhost:3000
```
